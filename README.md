# Respawn Claude Code Plugins

This repository hosts the Claude Code plugin marketplace for Respawn.

## Install marketplace

In Claude Code:
```
/plugin marketplace add respawn-llc/claude-plugin-marketplace
```

## Install ksrc plugin

```
/plugin install ksrc@respawn-tools
```

## Install flowmvi plugin

```
/plugin install flowmvi@respawn-tools
```

## Install firefox-cli plugin

```
/plugin install firefox-cli@respawn-tools
```

## Plugin list

- `ksrc`: skill for searching/reading Kotlin dependency sources
- `flowmvi`: FlowMVI skill for Kotlin Multiplatform MVI framework
- `firefox-cli`: Firefox control CLI skill for AI agents
